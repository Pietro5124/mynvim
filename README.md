# Windows
## Steps 
### 01 - Install Neovim 
    go to https://neovim.io and install nvim for windows 
### 02 - Put the ...\Neovim\bin in the path of the system
    go to system variables and edit the path of the system 
### 03 - Install chocolatery
    go to https://chocolatey.org and install chocolatery in powershell
### 04 - With the chocolatery install some packets 
    choco install ripgrep
    choco install fd 
    choco install zig
### 05 - Put the directory or git clone on the folders below
    {user}/AppData/Local/nvim
